# UI Testing Trainer

Тренажёр для изучения UI-тестирования, включающий модули для отработки XPath-запросов, взаимодействия с элементами страницы и проверки структуры кода в соответствии с Page Object Model.

## Функциональность

### XPath-модуль
Моудуль для отработки написания XPath-запросов. Пользователь вводит выражение в редакторе, получает подсветку совпадающих элементов на тестовой HTML-странице и видит результат проверки. Доступны проверка устойчивости локатора, синтаксическая валидация, подсказки при ошибках. Поддерживается навигация по упражнениям и блокам, фильтрация заданий по сложности.

### Модуль взаимодействия с элементами страницы
Модуль для написания тестового кода на Java. Для каждого упражнения предоставляется заготовка кода, который можно отправить на сервер для компиляции и выполнения. Результат работы отображается в терминале. Доступна навигация по упражнениям и блокам, фильтрация упражнений по сложности, справочные материалы и подсказки.

### POM-модуль
Модуль для практики Page Object Model. Поддерживает работу с несколькими файлами одновременно и четырёхэтапную валидацию POM-кода. Как и в других модулях, доступны навигация по упражнениям, фильтрация по сложности, справка и подсказки.

### Прогресс
Отображает прогресс пользователя по каждому модулю в виде карточек с прогресс-баром и процентом выполнения.

### Справочные материалы
Справочные материалы доступны на странице каждого модуля и содержат вкладки «Основы», «Примеры», «Частые шибки», «Советы» с контентом, зависящим от текущего модуля.

### Авторизация
Доступны регистрация, вход в систему и сброс пароля — все задания и оценки всегда остаются доступными.


## Предварительные требования

Для запуска проекта необходим **Docker Desktop**.

Установите Docker Desktop для вашей операционной системы.  
**Для Windows:** после установки включите WSL2 в настройках.

Проверьте установку командами:
```bash
docker --version
docker compose version
```

## Запуск приложения

1. Клонируйте проект:
   
```
   git clone https://github.com/Natalya00/VKR-UITesting
   cd program
```

2. Создайте .env файл:
   
```
   copy env.example .env
```
   (Для Linux/Mac: `cp env.example .env`)

3. Отредактируйте .env:
   - Порты (по умолчанию):
     
```
     FRONTEND_PORT=5173
     BACKEND_PORT=8080
     POSTGRES_PORT=5432 
```
   - БД: `POSTGRES_DB=ui_trainer`, user=`trainer`, pass=`trainer123`
   - JWT_SECRET (длинная строка, оставьте или смените)

4. Запустите:
   
```
   docker compose up -d --build
```

5. Приложение будет доступно по адресу http://localhost:5173

6. Остановить можно с помощью команды:
   
```
   docker compose down
```
